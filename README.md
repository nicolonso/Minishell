# Minishell

Project of 42 - This project is about creating a simple shell.

## Description

1️⃣ Read → Parse → Execute

A shell normally works like this:

User types command
      ↓
Shell reads it
      ↓
Shell parses it
      ↓
Shell executes it

Example:

$ ls -l | grep .c

Your shell must:

Read the line

Split it into tokens (ls, -l, |, grep, .c)

Execute the commands
